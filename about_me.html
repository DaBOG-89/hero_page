About Me.

Bio.

Hi, I’m Adesola Babaniji, a front-end developer based in Lagos. I specialize in creating responsive (perfect fit for laptops and smartphones), cost-effective, user-friendly websites using HTML, CSS, and JavaScript. I’m passionate about building web experiences that are accessible, intuitive, and visually engaging.

My Journey.

I discovered web development a few years ago and immediately fell in love with how code can bring ideas to life. Since then, I’ve dedicated myself to learning the fundamentals of front-end development and exploring back-end technologies like Node.js and PHP. Every project I work on teaches me something new and helps me grow as a developer.

Skills & Expertise.

  • Front-End: HTML, CSS, JavaScript, Bootstrap, Web3 CSS.
  • Back-End (Learning): Node.js, PHP, Python.
  • Database: MySQL, PostgreSQL.
  • Other: Responsive design, user experience (UX) principles, version control with Git.

Personal Projects.

My Mission.

I believe that the web should be accessible and user-friendly for everyone. My goal is to create projects that not only look good but also function seamlessly, giving users a smooth and enjoyable experience.
